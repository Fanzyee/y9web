Developing Ideas

Introduction

After analyzing multiple websites that propose a solution to my problem, I have a form of what my website needs to look like in order to provide a personal solution. My website needs to have features of innovation and interaction to connect with the user through the interface. I will design an aesthetically pleasing website, while maintaining the quality of the interactive features.

Design Specifications and Success Criteria

Interactivity

1. Website must include a HTML5 canvas graph that can:

            a) Display multiple countries

            b) Display data in multiple styles (pie chart, bar graph, world graph... etc.)

            c) Animate and display excess information when mouse hovers over each subsection of the graph

Ergonomics

2. Website must include filter functions, dropdowns, and autocompletes that sort ".json" data including:

            a) Different countries

            b) Different times periods and dates

            c) External information relating to the topic of Cholera

3. Website must include an audio/visual aid to guide the user though the website options include:

            a) A programmed identity (Siri, Alexa) that acts as a tutorial for the website

            b) A search bar for "Help"

Aesthetics

4. Website must include a contrasting and pleasing colour scheme:

            a) Text and other programs should be readable

            b) The colour scheme should not combat the text or the graphs
5. The use of space should not be clumped up; website should look modern, dynamic, and minimalist

Information

6. Website must be able to display information and statistics in a specific way:

            a) Basic information must be displayed automatically on the website

            b) More specific and external information will be found when the user completes a certain action (e.g. hovering the mouse, clicking, searching, ... etc.)
